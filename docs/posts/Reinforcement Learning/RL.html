What is Reinforcement Learning

Reinforcement Learning is learning what to do - how to map situations to actions - so as to maximize a numerical reward signal.

Reinforcement Learning is about learning the optimal policy (a mapping from states to actions) that maximizes the expected cumulative reward over time. In RL, the agent interacts with an unknown environment (typically modeled as an MDP) and learns from feedback (rewards) to make better decisions.

In many RL problems, the agent doesn’t have full knowledge of the MDP (e.g., the transition or reward functions) and must learn about the environment through trial and error.

In Reinforcement Learning (RL), several key definitions help describe how agents interact with the environment to learn optimal policies.

Value Function

The value function represents the expected cumulative reward an agent can obtain from a particular state, following a certain policy. The agent have two types of the value function:

Action-Value Function

The expected return starting from state s, taking action a, and following policy π thereafter. Mathematically: \[ Q_{\pi}(s,a) = \sum_{t'=t}^{T}\mathbb{E_{\pi_{\theta}}}[r(s_{t'}, a_{t'}) | s_{t}, a_{t}] \]

State-Value Function

The state-value function is defined as the expected return (cumulative reward) starting from state s and following policy π, which is also the expected value of action-value. Mathematically: \[ \begin{split} V_{\pi}(s_{t}) &= \sum_{t=0}^{T} \mathbb{E_{\pi_{\theta}}}\left[ r(s_{t'}, a_{t'}) | s_{t} \right] \\ &= \mathbb{E}_{a_t \sim \pi(a_{t} | s_{t})}[Q_{\pi_{\theta}}(s_{t}, a_{t})] \\ \end{split} \]

Objective Function

The Objective value of th RL algorithm is the state-value function of the initial value. And the object is to find a policy that maximize this value. The Objective function is defined as \(J(\theta) = V_{\pi}(s_{1})\) \[ J(\theta) = \mathbb{E}_{\tau \sim p_{\theta}(\tau)}\left[ \sum_{t} r(s_{t}, a_{t}) \right] \]

We want to find the parameter \(\theta^{*}\) that maximize the value of \(J(\theta)\). \[ \theta^{*} = \underset{\theta}{\operatorname{arg\max}} \quad J(\theta) \]

The above objective function is assumed that the tranjectory is finite, but what if the trajectory is infinite. The probabilistic graph model of MDP is described as:

However, we can group state\(s_{t}\) and action \(a_{t}\) together to formalize the Markov chain: where we marginal the state and action. \[ p(s_{t+1}, a_{t+1} | (s_{t}, a_{t})) = p(s_{t+1}| s_{t}, a_{t})\pi_{\theta}(a_{t+1} | s_{t+1}) \]

So the objective function become: \[ \underset{\theta}{\operatorname{arg\max}} \sum_{t=1}^{T}\mathbb{E}_{(s_{t},a_{t}) \sim p_{\theta}(s_{t},a_{t})}[r(s_{t}, a_{t})] \]

So, for the infinite horizon case, if we get the stationary distribution, we can have: the solution: Stationary Distribution: probability distribution over the states of the environment that remains unchanged over time, provided the agent follows a particular policy.

Reinforcement Learning is all about Expectation. Expected value can be continuous in the parameters of the corresponding value distributions even when the function we take expectation of is itself highly discontinuous. That why reinforcement learning can be use smoothing optimization methods such as gradient ascent to optimize objectives that are seemingly non-differentiable like binary rewards (win or lose). \(r(x)\) not smooth, and the \(\pi_{\theta}(a = \text{fall}) = \theta\), the expectation of \(\mathbb{E}_{\pi_{\theta}}[r(x)]\) is smooth in \(\theta\) The non-smooth, non-differentiable value under smooth, differentiable probability distributions are smooth and differentiable.

Algorithms

There are several algorithms used to solved the RL problems. The graph below, can be view as anatomy of all reinforcement algorithms:

  • Generate Samples: According to how generate samples from the policy, the algorithm can be devided into:
    • On-Policy: the samples is generate from the policy that we are trying to optimize
    • Off-Policy: the samples are generated from the policy(Behaviour Policy) different policy(Target Policy) that we trying to optimise.
  • Fit a model / Estimate the return: According to the algorithms we used, the algorithms can be devided to:
    • Value-Based Methods: we try to estimate the state-value function or action-value function. For example, Deep Q-Learning
    • Policy-Based Methods: we try to learning the policy directly without estimate the value functions. Besides those two types, we can also have a Model-Based methods, that we learn the model of the environment, then use the model to decide the optimal policy.
  • Improve the policy: Different methods has different way to improve the policy:
    • Gradient Ascent: We improve the gradient directly
    • RL By backprop: This is the model based RL that we backpropo through \(f_{\phi}\) and \(r\) to train \(\pi_{\theta}(s_{t}) =a_{t}\)
