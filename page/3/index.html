Hadoop 安装

单机

Java 安装

Hadoop 下载

  1. 官网查看安装包 binary download

  2. 下载 wget

  3. 注意！！！

    安装在 cd /usr/local/hadoop/ 目录下

    wget https://www.apache.org/dyn/closer.cgi/hadoop/common/hadoop-2.10.1/hadoop-2.10.1.tar.gz 是从官网下载，如果，返回的是

    说明在的不是压缩包，用

    tar -zxvf hadoop-2.10.1.tar.gz

    会报错

    gzip: stdin: not in gzip format
    tar: Child returned status 1
    tar: Error is not recoverable: exiting now

    因此，使用清华镜像安装

    https://mirrors.tuna.tsinghua.edu.cn/apache/hadoop/common/hadoop-2.10.1/

  4. 解压

  5. 将Hadoop的目录，导入环境变量

    export 导出的环境变量是临时的，关机就会失效，要想永久有效，需写入

    1. /etc/profile 变量中 然后 source /etc/profile使的修改生效。 改方法对所有用户有效，且是永久的
    2. 在 ~/.bash_profile 然后 source ~/.bash_profile
    3. 在 ~/.bashrc 中写入 然后 source ~/.bashrc

    ![Screenshot 2022-04-04 at 10.59.03](/Users/zhangyuyang/Desktop/blog/source/_posts/杂记/Screenshot 2022-04-04 at 10.59.03.png)

  6. 配置JAVA_HOME

    找到JDK所在的路径

    ![Screenshot 2022-04-04 at 11.08.26](/Users/zhangyuyang/Desktop/blog/source/_posts/杂记/Screenshot 2022-04-04 at 11.08.26.png)

    找到 JDK的路径： /etc/java/java-11-openjdk/java-11-openjdk-11.0.13.0.8-4.el8_5.x86_64

注意事项

当Hadoop运行时，会对外网打开端口，可能会被植入木马。所以最好改变Hadoop的默认端口
